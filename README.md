#  Automation Project For WalletHub  



## Getting Started



## Prerequisites
 
- Java Development Kit (JDK)
- Android SDK
- Node.js
- Webdriver
- Maven
- IntelliJ IDEA
- Android Device

## Instructions 
 Real android device required to perform tests, Please set desired capablitlies of your device in utility.BaseTest class. 


## Running the tests
    mvn clean install 
    RunTest.java
